<div align="center">
  <h2>🚀 Rapid Deployment & Quick Start</h2>
  <p>Get the entire full-stack application running on your local machine in under 60 seconds.</p>
</div>

### 1. Clone the Repository
```bash
git clone https://github.com/sathishr-ai/student-academic-analytics-platform.git
cd student-academic-analytics-platform
```

### 2. Initialize Virtual Environment (Recommended)
```bash
python -m venv venv

# Windows Activation:
venv\Scripts\activate

# macOS/Linux Activation:
source venv/bin/activate  
```

### 3. Install Core Dependencies
```bash
pip install -r requirements.txt
```

### 4. Launch the Application Server
```bash
streamlit run app.py
```
> 💡 *The application will automatically ignite and open your default web browser at `http://localhost:8501`. If it's your first time running, the system will automatically seed your SQLite database with 1,500 demo records!*
